AIRE : a generator of interactive web atlases

AIRE is a web application tool to create and manage rich atlases of thematic maps.

Built with PHP, the software generates the maps using PostgreSQL/PostGIS to query the geographic and statistical data, and UMN MapServer to render the maps.

Any thematic data

Demography, economy, environment... Once in the PostgreSQL database, anything can be rendered on the maps.

Multi-representation

Any dataset is rendered using multiple representation methods such as chloroplethe, proportional symbols, discontinuity borders, cartograms, grids and smooth maps.

Multiple levels of administrative units

For every scale, the complete set of maps is generated. For instance, country+region+departements data would make 3 maps for every enabled representation method, for a given dataset.

Scientific quality

AIRE is built by scientists and is made to be reliable. It provides:

  • Support for multiple projections (from the EPSG catalog)
  • Proper, complete legend management, in sync with the actual data
  • Proper map layouts (North arrow, lon/lat lines and others...)
  • Multi-language support for atlas/map titles, legends, units...
